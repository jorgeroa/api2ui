Create a top bar layout component that positions parameters in a responsive grid above results.

**Props:**
```typescript
interface TopBarLayoutProps {
  parameters: React.ReactNode
  results: React.ReactNode
  className?: string
}
```

**Layout structure:**
- Flexbox column layout
- Parameters panel: at top, border-bottom, responsive grid for parameter fields
- Results panel: below, flex-1, overflow-y-auto

**CSS for parameters grid (from RESEARCH.md):**
Use CSS Grid with auto-fit for responsive columns:
```css
gridTemplateColumns: 'repeat(auto-fit, minmax(240px, 1fr))'
```

This creates 2-3 columns depending on viewport width, adapting to parameter count.

**Tailwind classes:**
```
Container: flex flex-col min-h-0
Parameters: border-b border-gray-200 p-4

Results: flex-1 overflow-y-auto p-4
```

**Decisions honored (from CONTEXT.md):**
